git 作为一个版本控制工具，被程序员们广泛的使用，公司从去年开始也陆续将版本控制系统从 Perforce 向 git 转移。[《Pro Git》](https://git-scm.com/book/en/v2)作为官方推荐的学习 git 书籍，不可不读，而且该书已经有[中文版](https://git-scm.com/book/zh)。这篇就将记录自己学习使用 git 的过程。
